# Purpose

# Persona A — 「自然農に憧れる会社員」

名前（仮）: さとみ、34歳

職業: IT系会社員（フルリモート・東京在住）

---

## 価値観

```
・消費より体験を大切にしたい
・自分の食べるものは自分で作りたい気持ちがある
・自然・土・植物に癒しを感じる
・お金より「豊かさ」を追いたい
・「半農半X」「スローライフ」に憧れている
```

---

## 日常行動

```
・週末は山・農園・自然系のカフェへ
・週1〜2回ヨガかハイキング
・YouTube で「自然農」「パーマカルチャー」を検索して見る
・Instagram で田舎暮らし・農的暮らしのアカウントをフォロー
・note で自然農・移住・暮らし系の記事を読む
・「いつか農地を借りたい」と思っているが一歩踏み出せていない
```

---

## 悩み

```
・農業を始めたいが何から始めれば良いかわからない
・農業体験に行っても「管理されすぎた農業」に物足りなさを感じる
・自然農を実践している人と繋がる場所がわからない
・土地の見方がわからない。農地を見ても何もわからない
・移住・農業を始めるリスクが大きく感じてためらっている
```

---

## 憧れ

```
・休日に自分の農地を歩いている自分
・雑草を見て土の状態がわかるようになること
・「土地を読める人」になること
・自然の中で食べるものを作りながら生きること
・アナスタシア的な暮らしへの漠然とした憧れ
```

---

## SNS利用状況

```
Instagram: 毎日チェック。農的暮らし・自然・食系アカウントをフォロー
YouTube:  週2〜3時間。自然農・パーマカルチャー・田舎暮らし動画
Pinterest: たまに使う。農地・里山・古民家の画像を保存
note:     週1〜2本読む。農業参入・移住・自然農テーマ
X（旧Twitter）: 情報収集のみ
```

---

## このアプリを使う理由

```
「自然農に興味があるけど、土地の見方が全くわからない」
「農業体験より、土地を眺めながら妄想する時間が好き」
「仕事の合間に、自然農の世界に触れられる場所がほしい」
「土地を見る力がつけば、いつか農地を借りる決断ができそう」
```

---

## 継続利用理由

```
・通勤・休憩時間に「今日の土地」を見るのが習慣になる
・カードを解くたびに「少し土地がわかってきた」感覚が積み重なる
・週末の散歩中に「あの雑草、見たことある」と繋がる体験が起きる
・「こんな土地を持ちたい」という具体的な妄想が育つ
```

# Persona B — 「家庭菜園をもっと深くやりたいアラフォー」

名前（仮）: けんじ、42歳

職業: 中学校の教師（2年前からベランダ菜園→貸し農園を始めた）

---

## 価値観

```
・子どもたちに「食べ物がどこから来るか」を伝えたい
・農薬を使わずに育てることへの関心が高まっている
・自分の農業に「哲学」がほしい
・DIY・自分でできることへの喜びがある
・効率より「プロセス」を大切にする
```

---

## 日常行動

```
・週末に貸し農園で3〜4時間作業
・YouTube で「自然農」「不耕起」「有機栽培」を検索
・農業書・家庭菜園本を月1〜2冊読む
・農業SNSアカウント（X/Instagram）で情報収集
・近隣の農家に話しかけて教えてもらう機会を探している
```

---

## 悩み

```
・貸し農園の土地の特性がよくわからず、毎年試行錯誤
・雑草を「除草すべきもの」と思っているが、本当にそうなのか疑問
・農薬を使いたくないが、病害虫への対処法がわからない
・近くに相談できる自然農実践者がいない
・「土の状態」を見て判断できる目を育てたい
```

---

## 憧れ

```
・雑草を見て土の状態がわかる農家になること
・「この土地に何が合うか」がわかるようになること
・将来的に自然農で独立する夢
・子どもたちを農地に連れて行き、土地の見方を教えること
```

---

## SNS利用状況

```
X（旧Twitter）: 農業アカウントをフォロー・情報発信もたまにする
YouTube: 週3〜5時間。農業・自然農・有機農業動画を見る
Instagram: 家庭菜園の記録を投稿している
Facebook: 地域コミュニティや農業グループに参加
```

---

## このアプリを使う理由

```
「農地を毎週見ているのに、土地の読み方を誰にも教わっていない」
「雑草への見方を変えたい。除草すべきかどうか判断できるようになりたい」
「隙間時間に土地の見方を練習できる場所がほしい」
「農業の哲学（自然農・不耕起等）を体系的に理解したい」
```

---

## 継続利用理由

```
・農地に行ったとき「あ、このカードで見た雑草だ」と繋がる
・解説を読むたびに自分の農地への視点が変わる
・思想カードで農業哲学の理解が深まる
・カードが積み上がることで「成長している実感」がある
```

# 最優先ターゲット

## Persona A — 自然農に憧れる会社員（さとみ、34歳）

---

## 選定理由

### 1. 最もアプリの世界観と重なる

Persona A は「土地を眺めながら妄想したい」という欲求がすでにある。
農業学習アプリを求めているのではなく、
土地との感情的なつながりを求めている。
Imaging Agriculture の体験設計と完全に一致する。

---

### 2. SNSで最も拡散する

Instagram・Pinterest・note の利用頻度が高く、
「好きなものをシェアする」文化がある。
初期ユーザーが口コミ・投稿で次の100人を連れてくる確率が最も高い。

---

### 3. 課金への転換可能性が高い

月300円は Persona A の価値観では「安い」。
自然農・パーマカルチャー本は1,500〜3,000円。
土地を読む力を育ててくれるアプリに月300円は払える。

---

### 4. 数が多い

「自然農に憧れているが農業未経験の会社員」は、
日本に少なくとも数十万人いると推定される。
農家より圧倒的にリーチできる市場規模がある。
