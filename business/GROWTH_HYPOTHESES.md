# チャネル評価

## Pinterest

獲得難易度: ★★（低）
相性: ★★★★★（最高）
想定CPA: ほぼゼロ
想定拡散力: ★★★★（高）
優先度: 🔴 最優先

---

### なぜ相性が良いか

Imaging Agriculture のコアコンセプト（美しい土地写真を眺めながら妄想する）が
Pinterest の利用動機と完全に一致する。

```
Pinterestユーザーが保存するもの:
  里山の写真 / 古民家の農地 / 農的暮らしの画像 / 田んぼの朝
  ↓
これは Imaging Agriculture の Land Card そのもの
```

### 戦術

```
1. 土地カードの写真を Pinterest ピンとして投稿
2. 問いを添える: 「この土地なら何を育てたいですか？」
3. アプリへのリンクを貼る
4. 週3〜5枚のペースで投稿
5. 「里山」「自然農」「古民家」「農的暮らし」ボードで配布
```

### 仮説

Pinterest からの流入ユーザーは
「眺めることが好き」な層であり、
Imaging Agriculture の体験と最も親和性が高い。
D7 Retention が他チャネルより高くなる仮説を持つ。

## note

獲得難易度: ★★（低）
相性: ★★★★（高）
想定CPA: ゼロ
想定拡散力: ★★★（中）
優先度: 🔴 最優先

---

### なぜ相性が良いか

note のターゲット読者（農業・移住・自然・自己成長）と
Persona A・B・C の属性が重なる。

### 戦術

```
1. Founder として「土地を読む」をテーマにした note 記事を書く
   例: 「スギナが広がる畑を見て、初めて土地と対話できた話」
       「雑草を除草するのをやめたら、農地の見え方が変わった」
       「農業未経験の自分が自然農を学んだ方法」
2. 各記事末尾に Imaging Agriculture を紹介
3. 月2〜4本のペースで継続投稿
4. 「農業」「自然農」「移住」「半農半X」タグを活用
```

### 仮説

note 記事からの流入は質が高い（読了・共感ベースの流入）。
Answer Completion Rate が Pinterest 流入より高い仮説を持つ。

## Instagram

獲得難易度: ★★★（中）
相性: ★★★★（高）
想定CPA: ゼロ〜低
想定拡散力: ★★★★（高）
優先度: 🟡 第2優先

---

### なぜ相性が良いか

Persona A・C の主要 SNS。
農的暮らし・里山・自然農ジャンルのコミュニティがある。

### 戦術

```
1. 土地カードの写真をリール・フィードに投稿
   フォーマット: 写真 → 問い → 推奨回答の流れをリールにする
2. ストーリーズで「この土地に何を植える？」インタラクション
3. 農的暮らし・自然農インフルエンサーとのコラボ（初期は小規模でOK）
4. ハッシュタグ: #自然農 #農的暮らし #半農半X #里山 #土地を読む
```

### 仮説

リール形式は「土地の写真 + 問い + 解説」の3スライド構成が
最も Imaging Agriculture の体験に近く、
バイラルしやすいフォーマット。

## YouTube Shorts

獲得難易度: ★★★（中）
相性: ★★★（中）
想定CPA: ゼロ〜低
想定拡散力: ★★★★（高）
優先度: 🟡 第2優先

---

### 戦術

```
1. 「この土地に何を植える？3つの視点で読む」シリーズ
   60秒以内の縦型動画
2. 「スギナが多い土地に来た。あなたはどう見る？」
   実際の農地で撮影・解説
3. 「雑草で土地を読む方法」シリーズ
4. アプリ紹介は動画の最後15秒のみ（押しつけない）
```

### 仮説

YouTube の農業・自然農ジャンルはサブカルチャーだが
熱狂度が高い。
一本がバイラルすると急速に拡散する可能性がある。

獲得難易度: ★（最低）

想定CPA: ゼロ
想定拡散力: ★★（低）
優先度: 🟡 初期の直接獲得に活用

---

### 戦術

```
1. 自然農・有機農業のFacebookグループに参加
2. 「土地カードを一緒に試してほしい」と直接依頼
3. 感想をインタビュー → CUSTOMER_TRUTH_REPOSITORY.md に記録
```

### 注意

コミュニティへのスパム投稿は禁止。
まず貢献（価値ある投稿）をしてから紹介する。

# グロース仮説 Top 5

```
仮説1: Pinterest → アプリの流入が最もリテンション高い
  検証: チャネル別 D7 Retention を PostHog で計測

仮説2: 「土地を見る目が変わった」体験者が最も口コミを生む
  検証: 紹介経路を毎回ユーザーに確認する

仮説3: note 記事 1本の効果が SNS 投稿 10本分
  検証: 各チャネルの流入数を週次で比較

仮説4: 農業未経験者の方が継続率が高い
  検証: 農業経験の有無別に D30 Retention を比較

仮説5: 美しい写真カードのシェア率が最も高い
  検証: カード別のシェア数をPostHogで計測
```
