# Philosophy

お金より体験を先に作る。

体験の質が高ければ
課金は後からついてくる。

逆はない。

---

# Phase 1 — 完全無料（MVP〜MAU 100人）

期間: 最初の3〜6ヶ月

---

## 方針

完全無料。課金なし。

理由:

```
・体験価値の検証がまだ終わっていない
・100人以下で課金を設計しても意味がない
・「有料にすべきか」の答えはユーザーが教えてくれる
・この段階で課金障壁を作ると流入が止まる
```

---

## この段階で測るべきこと

```
□ 「もっと使いたい」という声が出ているか
□ 「課金してでも使いたい」という発言が自然に出るか
□ どのコンテンツへの需要が最も高いか
□ 無料でも D30 Retention 10%以上を達成できるか
```

# Phase 2 — 月300円プラン（MAU 100〜500人）

期間: 3〜12ヶ月目

---

## 月300円という価格の根拠

### 競合比較

| サービス | 月額 |
|---------|------|
| Duolingo Plus | 1,250円 |
| NHKまいにち○語 | 300円 |
| Kindle Unlimited | 980円 |
| Netflix（広告あり） | 790円 |
| ジム（最安） | 3,000円〜 |

300円は「コーヒー1杯」。
自然農に興味がある会社員（Persona A）にとって
「払うかどうか迷わない金額」。

---

### 月300円で成立する条件

```
□ 無料版で体験価値を感じている
□ 「もっと土地カードを見たい」という欲求がある
□ 「有料でしか見られないカードがある」理由が明確
□ 月300円 × 12ヶ月 = 3,600円の価値を感じられる
```

---

## 無料版 vs 有料版（案）

### 無料版

```
・月5枚の Land Card（ローテーション）
・基本的な解説
・プロフィール（Level / XP）
・連続記録
```

### 有料版（月300円）

```
・全 Land Card 無制限（追加されるたびに解放）
・Weed Cards（雑草カード）
・お気に入り保存
・観察ログ（自分だけのメモ）
・限定カード（季節・地域限定）
```

---

## 課金する理由（ユーザー視点）

```
「好きなカードをいつでも見返したい」
「雑草カードが解放されて、土地を読む幅が広がった」
「限定の季節カードを見たい」
「月300円なら、コーヒー1杯と同じ。払う価値がある」
「応援したいから課金する」（ファン課金）
```

---

## 課金しない理由（ユーザー視点）

```
「無料で十分」→ 無料版の制限を適切に設計する必要がある
「そこまで使っていない」→ Retention が低い = 体験価値が不十分
「高い」→ 300円は十分安いが、体験価値がないと払わない
「課金が面倒」→ Apple Pay / Google Pay で1タップで払える設計
```

# 無料のまま伸ばす案 vs 課金する案

## 案A: 完全無料 × 広告なし（寄付・支援型）

```
メリット:
  ・流入障壁ゼロ
  ・ミッション純粋性が保たれる
  ・「土地を見る文化を広げる」への共感が生まれやすい
  
デメリット:
  ・持続的な収益がない
  ・コンテンツ制作に限界が来る
  ・Founder が疲弊するリスク
  
向いている場合:
  ・NGO的な運営でも持続できる場合
  ・外部資金（補助金・投資）が得られる場合
```

---

## 案B: 月300円フリーミアム（推奨）

```
メリット:
  ・持続可能な収益モデル
  ・「払うほど価値がある」という品質基準が生まれる
  ・有料ユーザーはリテンションが高い傾向
  ・MAU 3,000人 × 転換率12% × 300円 = 月10万円超が見えてくる
  
デメリット:
  ・無料版の制限設計が難しい
  ・課金障壁で流入が一部減る可能性
  ・課金に値する体験品質が求められる
  
向いている場合:
  ・体験価値の検証が終わった後（Phase 2 以降）
  ・MAU 100人以上を達成してから
```

---

## 判定

**推奨: 案B（月300円フリーミアム）**

ただし Phase 1（〜MAU 100人）は完全無料。
「払いたい」という声が複数出てから課金設計を開始する。

課金のサインは
「有料でもいい」「もっとカードが欲しい」「応援したい」
という発言。
これが出始めたら課金設計を動かす。
