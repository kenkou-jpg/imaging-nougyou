# User Test Script

Imaging Agriculture

Version 1.0

---

# Purpose

「アプリが面白いか」を測るためではない。

「ユーザーの土地を見る視点が変化したか」
を検証するためのテスト。

---

# 対象ユーザー

優先順

1. 農業未経験 × 自然農・有機農業に興味がある人
2. 農業未経験 × 家庭菜園に興味がある人
3. 家庭菜園経験者（初心者〜中級）
4. 農業参入を考えている人

除外

* 農家（プロ）
* 農業の専門家
* テスターに慣れている人

---

# テスト時間

全体: 約30分

```
利用前インタビュー : 10分
MVP体験          : 10分
利用後インタビュー : 10分
（1週間後フォロー  : 15分 ← 別日）
```

---

# テスト進行のルール

```
□ 答えを誘導しない
□ ユーザーが黙っても待つ（最低10秒）
□ 「なぜ？」「もう少し教えてください」だけ使う
□ 「面白かったですか？」は聞かない
□ 録音・メモの許可を最初に取る
□ 「正解・不正解はない」と最初に伝える
```

---

# 1. 利用前インタビュー（10分）

## 目的

テスト前の「土地への視点の基準値」を記録する。

---

## 質問リスト

### 土地観察の習慣

```
Q1. 普段、農地や草地を意識して見ることはありますか？
    （見る / 見ない / たまに見る）

Q2. 散歩や旅行中に畑や空き地に目が行くことはありますか？
    もし行くとしたら、何を見ていますか？

Q3. 雑草を見るとき、何を感じますか？
    （邪魔 / 気にならない / 興味深い / その他）
```

---

### 農業・土地との関係

```
Q4. 農業にどのくらい興味がありますか？
    （全くない / 少し興味がある / かなり興味がある / 実践している）

Q5. 自然農・有機農業・草生農法などを聞いたことがありますか？
    知っていれば、どんなイメージですか？

Q6. 土地を「読む」という言葉を聞いてどんなイメージが浮かびますか？
```

---

### 自由

```
Q7. 農業や土地に関して、最近気になっていることや
    疑問に思っていることはありますか？
```

---

```
[Q6] 「土地を読む」へのイメージ:

メモ（表情・態度・特徴的な言い回し）:
```

# 3. 利用後インタビュー（10分）

## 目的

視点の変化の芽生えを確認する。

---

## 禁止質問

```
❌「何を学びましたか？」
❌「どんな知識が身につきましたか？」
❌「正解できましたか？」
❌「面白かったですか？」
```

---

## 質問リスト

### 体験した土地について

```
Q1. 今日見た土地の中で、印象に残っているものはありますか？
    それはなぜですか？

Q2. 気になった土地はありましたか？
    実際に見に行きたくなった場所はありますか？

Q3. 見ていて「行ってみたい」と思った土地はありましたか？
```

---

### 視点の変化

```
Q4. 雑草や草地への見方は何か変わりましたか？
    （変わった / 変わっていない / 少し変わった気がする）

Q5. 「土地を読む」という感覚、少し分かってきましたか？
    どんな感覚でしたか？

Q6. 今後、散歩中や外出中に農地や草地を見てみたくなりましたか？
    （なった / ならなかった / もともと見ていた）
```

---

### 体験の感想

```
Q7. 使っていて「もっとこうだったらな」と思ったことはありますか？

Q8. もし友人に紹介するとしたら、どんな人に紹介しますか？
    なんて言いますか？
```

---

# 4. 1週間後フォローアップ（15分・別日）

## 目的

最重要。

アプリを閉じた後の「行動変化」を確認する。
視点の変化が日常に持続したかを測る。

---

## 実施方法

LINE / メール / 電話 / 対面のいずれか。
相手の都合に合わせる。

---

## 質問リスト

### 行動変化の確認

```
Q1. この1週間で、農地や草地を意識して見た場面はありましたか？
    （あった / なかった）
    あった場合：どんな場所でしたか？

Q2. 散歩中・通勤中・旅行中に雑草や草地が目に入ることはありましたか？
    そのとき何を考えましたか？

Q3. アプリで見た土地や雑草を、実際に外で思い出したことはありましたか？

Q4. 土地を見て「ここなら何を育てたいか」と考えることはありましたか？
```

---

### アプリの継続

```
Q5. この1週間でアプリを開きましたか？
    （開いた: __回 / 開かなかった）
    開かなかった場合、理由は何ですか？

Q6. アプリを開いていない時でも、アプリのことを思い出したことはありましたか？
```

---

### 変化の実感

```
Q7. 1週間前と比べて、土地・雑草・農業への興味や視点に
    何か変化を感じますか？

Q8. このアプリを使い続けたいと思いますか？
    その理由は何ですか？
```

---

## 記録テンプレート（1週間後）

```
参加者ID: P__
フォローアップ日: YYYY-MM-DD

[Q1] 農地・草地を意識して見たか:
[Q2] 散歩中に雑草に目が行ったか:
[Q3] アプリを実物で思い出したか:
[Q4] 「何を育てたいか」を考えたか:
[Q5] アプリ利用回数:
[Q6] アプリ閉じた後に思い出したか:
[Q7] 視点の変化の実感:
[Q8] 継続意向:

行動変化スコア（0-3）:
  0: 何も変化なし
  1: 少し気になるようになった
  2: 見る機会が増えた
  3: 積極的に見に行くようになった

総合評価: 成功 / 部分成功 / 要改善
```

# Success Signal 一覧

以下の発言が出たとき、即座に記録する。

## 強い成功シグナル（★★★）

```
「旅行中に畑を見るようになった」
「雑草を見るのが楽しくなった」
「土地を見る目が変わった」
「ここなら何を育てたいか考えるようになった」
「外に出てこういう場所を探してみたくなった」
「写真を見て、実際にその場所に行きたくなった」
```

---

## 中程度の成功シグナル（★★）

```
「散歩中に草地が気になるようになった」
「農地の前で立ち止まるようになった」
「以前は見ていなかった草が目に入るようになった」
「また使ってみたい」
「友人に見せたい」
```

---

## 弱い成功シグナル（★）

```
「面白かった」（何が面白かったかを掘り下げる）
「写真が綺麗だった」（視点変化の有無を確認）
「また開くかもしれない」
```

---

# Failure Signal 一覧

以下の発言は失敗シグナル。記録して原因を分析する。

```
「勉強になりました」→ 学習アプリになっている
「知識が増えました」→ 知識を得ることが目的化されている
「クイズみたいで楽しかった」→ ゲーム化されすぎている
「正解できて嬉しかった」→ 正解することが目的化されている
「難しすぎた」→ 問いの難易度を下げる
「何を答えればいいかわからなかった」→ 問いが不明確
「写真が何なのかわからなかった」→ 写真の説明が必要
```

# 運用方法

## 5人テスト版（Phase 0 / MVP直後）

目的: 根本的な体験価値の検証

```
人数: 5人
方法: 対面 or ビデオ通話（観察のため）
実施期間: 1週間以内に5人完了
フォロー: 全員に1週間後フォローアップ
記録先: CUSTOMER_TRUTH_REPOSITORY.md

判定基準:
  ★★★シグナル 1件以上 → 続ける価値あり
  ★★シグナル 3件以上 → 方向性は正しい
  失敗シグナルが3件以上 → 根本的見直し
```

---

## 10人テスト版（Phase 1 前半）

目的: 初期仮説の精緻化とコンテンツ改善

```
人数: 10人
方法: 対面 + リモート混在 OK
実施期間: 2週間
フォロー: 7人以上に1週間後フォローアップ
記録先: CUSTOMER_TRUTH_REPOSITORY.md

追加観察:
  - どのカードが最も時間をかけて見られるか（PostHog）
  - どの選択肢が最も選ばれるか
  - 解説のどの文章で表情が変わるか

判定基準:
  行動変化スコア平均 1.5以上 → Phase 1 継続
  行動変化スコア平均 1.0未満 → 体験設計を見直し
```

---

## 20人テスト版（Phase 1 後半 / Phase 2 前）

目的: スケール前の品質確認とリテンション検証

```
人数: 20人
方法: リモート主体 + 定量データ活用
実施期間: 3〜4週間
フォロー: D1・D7・D30 全員追跡

追加測定:
  - D30 Retention 率
  - Knowledge Unlock Rate
  - Streak 継続率
  - NPS スコア（推奨度）

判定基準:
  D30 Retention 10%以上 → Phase 2 へ進む
  ★★★シグナル累計 5件以上 → コミュニティ機能の検討開始
  NPS 40以上 → 口コミによる成長フェーズへ
```
