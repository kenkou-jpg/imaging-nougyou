# Brand Guidelines

Imaging Agriculture

Version 1.0

# ブランド人格

## 目指す人格

里山を案内してくれる友人

---

この友人は

* 農業の先生ではない
* 資格を持った専門家でもない
* 農業書を暗記した人でもない

---

この友人は

* 里山が好きで、よく一人で歩いている
* 雑草の名前より「この草が生える理由」を考えている
* 「正解はわからないけど、こう考えると面白い」と言う
* 押しつけがましくなく、でも深い視点を持っている
* 一緒にいると、なぜか自分も土地が気になってくる

---

## 避ける人格

```
専門家   → 難しい言葉で説明する
講師     → 正解を教えてくれる
試験官   → 正解・不正解を判定する
コーチ   → 目標達成を管理する
SNSインフルエンサー → フォロワーに語りかける
```

# 文章ルール

## 基本原則

説明するより、想像させる。

---

## 文章のリズム

短い。

一文が長いと、眺める時間が消える。

---

## 問いの形

答えを言わずに、問う。

悪い例（説明）

```
スギナは酸性土壌を示します。
不耕起栽培に向いた農地です。
```

良い例（問い）

```
スギナが広がるこの土地。
あなたは何を想像しますか？
```

---

## 解説の書き方

答えを教えるのではなく、視点を渡す。

悪い例

```
正解はAです。
この土地は排水性が悪く、果樹には向きません。
```

良い例

```
スギナが多い土地は、土が締まっていることが多い。
焦って植えるより、まず一年、この土地を見守ってみるのも面白い。
```

---

## 一人称

「あなた」は使う。
「皆さん」は使わない。

このアプリは、
あなたと土地の一対一の体験。

---

## 語尾

断定しすぎない。

「〜かもしれない」
「〜と考えることもできる」
「〜という読み方もある」

農業に正解はない。
言葉もそれを反映する。

---

## 数字・専門用語の扱い

pH・土壌微生物・菌根菌などの用語を使う場合は
必ず感覚的な言葉に変換する。

悪い例

```
pH4以下の強酸性土壌のため、
石灰投入によるアルカリ化が推奨されます。
```

良い例

```
とても酸っぱい土。
植物が嫌がっているかもしれない。
石灰を少し足すと、土がほっとするかもしれない。
```

# ネーミング・ラベル

## アプリ内テキスト方針

```
「クイズ」ではなく「土地カード」
「正解」ではなく「推奨回答」
「学習履歴」ではなく「土地の記録」
「ポイント」ではなく「XP」（説明不要な表記）
「次の問題」ではなく「次の土地へ」
「間違い」ではなく「別の見方もある」
```
