# Emotional Design

Imaging Agriculture

Version 1.0

---

# 目的

知識を増やすことではない。

視点を変えること。

---

成功状態

旅行中に畑を見る。
散歩中に雑草を見る。
空き地を見て
「ここなら何を育てたいだろう」
と考えるようになること。

---

# ユーザー感情マップ

## 利用前（初回インストール時）

### 想定されるユーザーの状態

```
「農業って興味はあるけど、どこから始めればいいか」
「自然農が気になって検索してたら見つけた」
「農地を借りようか悩んでいる」
「単純に植物・土・自然が好き」
```

### この段階での感情

```
期待: 「面白そう」
不安: 「難しそう・農業知識がないと使えないかも」
好奇心: 「土地を読むって何？」
```

### デザインが答えるべき問い

「知識がなくても楽しめますか？」
→ 最初のカードで答える。難しい用語を使わない。

---

## 利用中（1回目のセッション）

### ホーム画面を開いた瞬間

```
目標感情: 「あ、綺麗な写真。この土地気になる」
避ける感情: 「どこをタップすればいいか複雑」
            「いきなり登録を求められる」
```

デザイン判断:
→ 初回は写真全面 + シンプルなCTAのみ
→ ログイン誘導は2回目以降

---

### 質問画面を見たとき

```
目標感情: 「うーん、なんだろう。眺めてみよう」
避ける感情: 「急いで答えなきゃ」
            「知識がないと答えられない」
```

デザイン判断:
→ タイマーなし
→ 選択肢は農業知識不要でも直感で選べる問い
→ 地図・写真を自由に拡大できる

---

### 回答した瞬間

```
目標感情: 「どんな読み方があるんだろう」
避ける感情: 「正解した！やった！（過剰な達成感）」
            「不正解だ、恥ずかしい（罰感）」
```

デザイン判断:
→ 正解・不正解の大きな演出なし
→ 「こんな見方もあります」のトーン
→ 推奨回答であり、正解ではない

---

### 解説を読んだとき

```
目標感情: 「なるほど、そういう読み方か。面白い」
           「今度外で確かめてみよう」
避ける感情: 「勉強した（義務感）」
            「覚えなければ（暗記感）」
```

デザイン判断:
→ 解説は「新しい視点の提案」として書く
→ 外に出たくなるひと言を最後に添える

---

### XP を受け取ったとき

```
目標感情: 「少し嬉しい。でも主役じゃない」
避ける感情: 「XPのためにやっている」
```

デザイン判断:
→ XP演出は控えめ（大きなアニメーション禁止）
→ 写真・解説の余韻を壊さない

---

## 利用後（セッション終了時）

### セッションを終えた直後

```
目標感情: 「気持ちよかった。また来よう」
           「次の土地も見てみたい」
           「今度外で土地を見てみよう」
避ける感情: 「やっと終わった」
            「もっとやらないといけない」
```

デザイン判断:
→ 「また来てね」の強制なし
→ 「今日見た土地」の振り返りを静かに表示
→ 「散歩に行きたくなった？」の軽いブリッジ

---

### アプリを閉じた後

```
目標状態:
  歩いているとき、草を見て
  「この草、あのカードで見たやつかも」
  と思い出す
```

これが最高の成功。
アプリを閉じた後にも影響が残る体験。

---

## 継続利用時（1週間後〜）

### 1週間後の感情目標

```
「また土地を見たくなってきた」
「旅行中に農地が気になるようになった」
「散歩中に雑草を気にするようになった」
```

---

### 1ヶ月後の感情目標

```
「土地の見え方が変わってきた気がする」
「以前は気にしなかった草地が気になる」
「ここなら何を育てたいかを自然に想像する」
```

---

### 3ヶ月後の感情目標

```
「農地を見ることが旅行の楽しみの一つになった」
「雑草を見ると嬉しくなる」
「農地を持ってみたいと考え始めた」
```

---

# 感情マップ総覧

```
時間軸          感情ピーク              デザイン担当要素

初回起動         「この写真、綺麗」       ホーム写真
                                        
質問中           「うーん、どれだろう」   写真の大きさ・余白

回答後           「そういう見方か！」     解説のトーン

セッション後     「また来よう」          静かな終わり方

アプリ外         「あ、これかも」        記憶に残る問いと写真

1週間後         「土地が気になる」       連続記録・推薦カード

1ヶ月後         「視点が変わった」       蓄積された土地の記録
```

# 感情設計の禁止事項

## 過剰な達成感の演出

```
大きな正解音
紙吹雪アニメーション
「すごい！完璧です！」の文言
連続正解ボーナスの強調
```

→ 達成感の主役は「土地を読めた感覚」。
  アプリのアニメーションではない。

---

## 恐怖・不安の活用

```
「今日やらないと連続記録が途切れます！」
「他のユーザーはもっと先まで進んでいます」
「この問題を正解できる人は〇%だけ」
```

→ 禁止。農業に競争心は必要ない。

---

## 不正解時の強い罰感

```
大きなバツマーク
「残念！」の大きな表示
不正解音（ブザー音）
```

→ 「推奨回答はこちらでした」で十分。
  罰感なし。

---

# 感情設計の成功基準

5人のユーザーテストで、以下の発言が1件でも出たら成功。

```
「旅行中に畑を見るようになった」
「雑草を見るのが楽しくなった」
「土地を見る目が変わった」
「ここなら何を育てたいか考えるようになった」
「外に出たくなった」
「また土地を見たくなってきた」
```

---

以下の発言が出たら再設計が必要。

```
「勉強になりました」
「クイズみたいで楽しかった」
「知識が増えました」
「正解できて嬉しかった」
```
