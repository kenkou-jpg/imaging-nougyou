# Visual Direction

Imaging Agriculture

Version 1.0

---

# Visual Concept

土と光と静けさ。

派手さはいらない。
農地の写真が語るから。

UIは「額縁」。
写真が絵画で、UIは額縁。
額縁は目立たない。

---

# 写真方針

## 写真が主役

写真は画面の50〜70%を占める。
テキストは後から添える。

写真を見ていると、
自然に「この土地に行きたい」と思えるものを選ぶ。

---

## 写真の品質基準

```
✅ 良い写真
  自然光（特に午前・夕方）
  土・草・空のどれかが主役
  季節感がある
  人がいない（または小さい）
  余白がある（空・地平線・霞）

❌ 悪い写真
  人工的な光
  農業機械が主役
  農薬袋・化学肥料が写っている
  説明的すぎる（テキストが写真に入っている）
  暗い・汚い印象
```

---

## 写真のムード

朝の光を浴びた草地。
霧の中の棚田。
秋の里山の斜面。
雨上がりの農道。

「ここに行ってみたい」
「この土地を歩いてみたい」
と感じる写真を選ぶ。

---

## 写真比率・サイズ

ホーム画面: 3:4（縦長・Pinterestカード）
カード詳細: 4:3〜1:1（横または正方形）
サムネイル: 1:1（一覧表示用）

---

## 写真の撮影ガイドライン（コンテンツ制作向け）

```
時間帯: 早朝 or 夕方（ゴールデンアワー）
天気:  晴れ・曇り・雨上がり（快晴は避ける）
アングル: 低め（地面に近い視点・人の目線より低く）
フォーカス: 前景にピント・奥はボケる
色温度: 暖色系（4000〜5500K）
```

---

# 余白方針

## 余白は呼吸

余白は「勿体ない空間」ではなく
「土地を眺める時間」。

テキストを詰め込むほど
眺める余裕が消える。

---

## 余白の量

```
カード内の余白: 最低 24px 四方
テキストと写真の距離: 最低 32px
ボタンの上下余白: 最低 20px
選択肢の行間: 最低 16px
```

---

## 空白を埋めない

「ここが寂しいから何かを入れよう」は禁止。

空白があるなら、
写真をもっと大きくする。

---

# レイアウト方針

## 縦スクロール一択

横スクロールは使わない。
土地を眺める動作は「縦」。

---

## カードレイアウト

```
┌──────────────────────────┐
│                          │
│    写真（画面高さ50%）     │
│                          │
│                          │
├──────────────────────────┤
│  タイトル                 │
│  難易度 ・ 土地タイプ     │
├──────────────────────────┤
│  問い                    │
├──────────────────────────┤
│  選択肢 A                │
│  選択肢 B                │
│  選択肢 C                │
│  選択肢 D                │
├──────────────────────────┤
│  [回答する]               │
└──────────────────────────┘
```

写真と問いの間に余白を挟む。
急がせない。

---

```

```

# UI原則

## 原則1 — 一画面一行動

ホーム: 「今日の土地を開く」
質問: 「回答する」
結果: 「次の土地へ」

一画面に二つの行動を求めない。

---

## 原則2 — ボタンは一つ

一画面でタップできる主要ボタンは一つ。
サブアクションは小さく。

---

## 原則3 — 色はシグナルに使う

色でムードを作らない。
色は「ここをタップしてほしい」だけに使う。

---

## 原則4 — アニメーションは控えめ

土地を感じさせるアニメーション: OK
達成を過剰に祝うアニメーション: NG

フェードイン・スライド: OK
バウンド・キラキラ・爆発: NG

---

# 色の方向性

## カラーコンセプト

土・葉・光・霧。

自然の中にある色を使う。

---

## プライマリカラー（仮方向性）

```
土の色    : 茶系ニュートラル（#7C6454 程度）
葉の色    : くすんだグリーン（#5A7A5C 程度）
光の色    : 温かいクリーム（#F5F0E8 程度）
```

# タイポグラフィ

## フォント方針

```
日本語: 游ゴシック or Noto Sans JP（Light〜Regular）
英数字: システムデフォルト or Inter
```

装飾的なフォント禁止。
読みやすさより、空気感を邪魔しないことを優先。

---

## テキストサイズ

```
タイトル（カード名）: 20〜24px / Bold
問いテキスト: 16〜18px / Regular
選択肢テキスト: 15〜16px / Regular
解説テキスト: 14〜15px / Regular / Line-height 1.8
補足情報: 12〜13px / Light
```

---

## 行間

読みやすさより「眺めやすさ」を優先。

解説テキストの行間は 1.8〜2.0。
詰め込まない。

---

## テキストカラー

```
メインテキスト: #2D2D2D（濃すぎない黒）
サブテキスト: #7A7A7A（グレー）
解説テキスト: #4A4A4A
背景: #F8F5F0（温かみのあるオフホワイト）
```

---

# アイコン・イラスト方針

## アイコン

シンプルなラインアイコン。
装飾なし・モノクロ基調。

農業・土地系のカスタムアイコンは
MVP では作らない。
システムデフォルト + ライブラリで対応。

---

## イラスト

MVP では使わない。
写真が主役のため、イラストで補完しない。

---

# Instagram vs Pinterest の違い

このアプリは Instagram 風ではなく Pinterest 風。

```
Instagram: リアルタイム・いいね・フォロー・自己表現
Pinterest: 保存・妄想・眺め・発見・後で見返す

→ Imaging Agriculture は Pinterest 寄り
  「保存したくなる」「後で見返したくなる」
  「眺めていたい」体験を目指す
```
